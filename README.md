# Proyecto WF - Bot de WhatsApp

Bot de WhatsApp automatizado usando whatsapp-web.js

## 📋 Requisitos

- Node.js v14 o superior
- npm o yarn
- WhatsApp activo en tu navegador

## 🚀 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/lorenzojoapa-sketch/proyecto-WF.git
cd proyecto-WF
```

2. Instala las dependencias:
```bash
npm install
```

3. Copia el archivo .env:
```bash
cp .env.example .env
```

4. Inicia el bot:
```bash
npm start
```

5. Escanea el código QR con WhatsApp

## 📝 Comandos

- `!hola` - Saluda el bot
- `!ayuda` - Muestra la ayuda

## 🛠 Desarrollo

Para modo desarrollo con reinicio automático:
```bash
npm run dev
```
